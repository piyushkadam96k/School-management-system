How to Use the School ERP App

This page describes the common workflows for administrators and teachers. Use this guide to get started quickly.

1. Login

  • Open /login. Use the default admin: admin / admin123.
  • Change the password in the database for production or create a new user account.

2. Dashboard

  • After login, the dashboard shows counts for Classes, Students, and Exams.
  • Use the navigation to go to Classes, Fees, and other sections.

3. Manage Classes

  • Navigate to /classes to view classes.
  • Admin users can add a class via /add_class, edit or delete classes from the class page.

4. Add Students & Subjects

  • From a class detail page, use + Add Student to add a student (name and roll number).
  • Use + Add Subject to add subjects to the class.

5. Exams & Entering Marks

  • Create an exam for the class at /exams/{class_id}. You may add a weight and type.
  • To enter marks for a student: open /enter_marks/{class_id}?exam_id={exam_id}, pick a student and save marks for each subject.

6. View Results & Export

  • View individual student results: /result/{student_id}?exam_id={exam_id}. Download PDF using the download link (ReportLab is required for PDF).
  • View class results: /class/{class_id}/results?exam_id={exam_id}. Export class results CSV through the Export CSV button.

7. Attendance

  • Navigate to /attendance/{class_id}. Pick a date and mark present students, then Save.

8. Fees

  • View a summary at /fees.
  • Open class-specific fees at /fees/class/{class_id} and add fee items or record payments for students.

9. Search

  • Use the search box in the header or go to /search?q=NAME_OR_ROLL to find a student.

10. Users & Roles

  • The app stores simple users in the database (admin/teacher). Only admin can add, edit, or delete classes/subjects and fee items.
